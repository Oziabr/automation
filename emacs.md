
|	              | **file**                 |
|---------------|--------------------------|
| **`C-x C-f`** |	open                     |
| **`C-x C-s`** |	save current             |
| **`C-x C-w`** |	save as                  |
| **`C-x s`**   |	save all                 |

|               | **cursor** | 
|---------------|------------|
| **`C-f`**     | >          |
| **`C-b`**     | <          |
| **`C-n`**     | v          |
| **`C-p`**     | ^          |
| **`M-f`**     | -> word    |
| **`M-b`**     | <- word    |
| **`C-a`**     | BOL        |
| **`C-e`**     | EOL        |
| **`M-a`**     | BOP        |
| **`M-e`**     | EOP        |
| **`C-v`**     | page down  |
| **`M-v`**     | page up    |
| **`M-<`**     | BOF        |
| **`M->`**     | EOF        |
| **`C-l`**     | recenter   |


|               | **buffer**            |
|---------------|-----------------------|
| **`C-x C-b`** | list                  |
| **`C-x b`**   | switch to named (tab) |
| **`C-x k`**   | kill                  |

|             | **window**           |
|-------------|----------------------|
| **`C-x 0`** |	close current window |
| **`C-x 1`** |	close others         |
| **`C-x 2`** |	split vertically     |
| **`C-x 3`** |	split horizontally   |
| **`C-x o`** |	switch               |

|               | **frame**    |
|---------------|--------------|
| **`C-x 5 0`** |	close        |
| **`C-x 5 1`** |	close others |
| **`C-x 5 2`** |	new          |

|---------------|------------|
| **`C-x C-c`** |	Exit Emacs |
